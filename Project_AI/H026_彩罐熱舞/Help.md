## 二、遊戲特色 GAME FEATURES

### 消除掉落特色 CASCADING FEATURE

▪ 遊戲中若達成連線，金框符號除外的連線符號將消除，並且會再從上方掉落新的符號。
ALL SYMBOLS IN WINNING COMBINATIONS WILL DISAPPEAR EXCEPT FOR THOSE WITH A GOLDEN FRAME, AND THE EMPTY POSITIONS WILL BE FILLED BY SYMBOLS FALLING DOWN FROM ABOVE.

▪ 消除掉落特色會持續發生，直到沒有連線為止。
THE FEATURE WILL COMPLETE WHEN THERE ARE NO NEW WINNING COMBINATIONS.

---

### 金框特色 GOLDEN FRAME FEATURE

▪ 金框符號只出現在第 2、3 和第 4 轉輪上。
THE GOLDEN FRAME SYMBOLS WILL APPEAR ON REELS 2, 3 AND 4 ONLY.

▪ 若達成的連線包含金框符號時，金框符號將在原位置轉換為百搭符號 [Wild]。
IF THE WINNING COMBINATION INCLUDES A SYMBOL WITH A GOLDEN FRAME, THE SYMBOL WILL BE CHANGED INTO A [WILD] AT ITS ORIGINAL POSITION.

▪ 在基本遊戲中，第 3 轉輪有機會隨機整條出現帶有倍數的金框符號。
IN THE BASE GAME, REEL 3 MAY RANDOMLY HAVE ALL SYMBOLS APPEAR AS GOLDEN FRAME SYMBOLS WITH MULTIPLIERS.

---

### 乘倍符號特色 MULTIPLIER SYMBOL FEATURE

▪ 金框符號有機會附帶倍數，隨機倍數值包含 2X、3X、5X、8X、10X、15X、20X、25X、30X、50X、80X 或 100X。
GOLDEN FRAME SYMBOLS MAY CARRY A MULTIPLIER. RANDOM MULTIPLIER VALUES INCLUDE 2X, 3X, 5X, 8X, 10X, 15X, 20X, 25X, 30X, 50X, 80X OR 100X.

▪ 每當帶倍數的金框符號參與得分，其倍數值將累積至盤面上方的倍數計數器中。
WHEN A GOLDEN FRAME SYMBOL WITH A MULTIPLIER CONTRIBUTES TO A WINNING COMBINATION, ITS MULTIPLIER VALUE IS ADDED TO THE ACCUMULATED MULTIPLIER DISPLAYED ABOVE THE REELS.

▪ 當消除掉落特色結束時，本次旋轉的總贏分將乘上累積的總倍數，完成結算。
WHEN THE CASCADING FEATURE ENDS, THE TOTAL WIN FOR THAT SPIN WILL BE MULTIPLIED BY THE ACCUMULATED TOTAL MULTIPLIER.

## 三、免費遊戲特色 FREE GAME FEATURE

▪ 當 3 個以上 [Scatter] 出現在任意位置，即可觸發免費遊戲特色。
3 OR MORE [SCATTER] APPEARING IN ANY POSITION WILL TRIGGER THE FREE GAME FEATURE.

▪ 點擊開始啟動免費遊戲特色。
CLICK THE START BUTTON TO START THE FREE GAME FEATURE.

▪ 在免費遊戲中，若再次出現 3 個以上 [Scatter]，可依相同規則追加免費場次。
DURING THE FREE GAME, 3 OR MORE [SCATTER] WILL AWARD ADDITIONAL FREE SPINS ACCORDING TO THE SAME RULES.

▪ 在免費遊戲中，第 3 轉輪的符號必定為帶有倍數的金框符號。
DURING THE FREE GAME, ALL SYMBOLS ON REEL 3 ARE GUARANTEED TO BE GOLDEN FRAME SYMBOLS WITH MULTIPLIERS.

▪ 在免費遊戲中，累積倍數不會在每場之間重置，將持續保留至整段免費遊戲結束後才清空。
DURING THE FREE GAME, THE ACCUMULATED MULTIPLIER IS NOT RESET BETWEEN SPINS AND CARRIES OVER UNTIL THE ENTIRE FREE GAME SESSION ENDS.

▪ 在免費遊戲中，投注與觸發免費遊戲特色時相同。
DURING THE FREE GAME, BET PER SPIN REMAINS THE SAME AS WHEN THE FREE GAME FEATURE IS TRIGGERED.

## 四、購買免費遊戲特色 BUY FEATURE

▪ 可通過 75X 總投注的金額來購買免費遊戲特色。
BUY FEATURE CAN BE PURCHASED BY USING A VALUE EQUAL TO 75X TOTAL BET.

▪ 購買免費遊戲特色後保證能觸發免費遊戲。
PURCHASING THE FREE GAMES FEATURE GUARANTEES FREE GAMES.

## 五、OP JACKPOT

▪ 在遊戲中，OP JACKPOT 特色由轉輪上的 [Scatter] 隨機觸發。
OP JACKPOT FEATURE IS TRIGGERED RANDOMLY BY [SCATTER] ON THE REELS.

▪ OP JACKPOT 特色會在畫面中出現 12 個 [彩罐]，其中包括 [GRAND]、[MAJOR]、[MINOR] 及 [MINI]。
DURING OP JACKPOT FEATURE, THERE ARE 12 [PIÑATA] WHICH CONSIST OF [GRAND], [MAJOR], [MINOR] AND [MINI].

▪ 選擇 1 個 [彩罐] 以揭示 [GRAND]、[MAJOR]、[MINOR] 或 [MINI]，當玩家獲得 3 個相同的 OP JACKPOT 符號即可獲得該彩金並結算總獎金。
SELECT A [PIÑATA] TO REVEAL [GRAND], [MAJOR], [MINOR], OR [MINI]. IF PLAYER GETS 3 SAME JACKPOT SYMBOLS, THE PLAYER WILL WIN THE JACKPOT AND CALCULATE TOTAL WINNINGS.

▪ [GRAND] 為連機累進彩金。單場遊戲中，投注選項選擇 {2.00} 以上即可解鎖 [GRAND]。
[GRAND] IS LINKED PROGRESSIVE JACKPOT. IF THE SINGLE GAME BET OPTION IS {2.00} OR ABOVE, THE PLAYER WILL UNLOCK [GRAND].

▪ [MAJOR] 為連機累進彩金。單場遊戲中，投注選項選擇 {2.00} 以上即可解鎖 [MAJOR]。
[MAJOR] IS LINKED PROGRESSIVE JACKPOT. IF THE SINGLE GAME BET OPTION IS {2.00} OR ABOVE, THE PLAYER WILL UNLOCK [MAJOR].

▪ [MINOR] 為紅利彩金，依投注選項改變。
[MINOR] WILL CHANGE IF THE PLAYER SELECTS DIFFERENT BET OPTION.

▪ [MINI] 為紅利彩金，依投注選項改變。
[MINI] WILL CHANGE IF THE PLAYER SELECTS DIFFERENT BET OPTION.

▪ 玩家投注越高，觸發 OP JACKPOT 特色的機會越高。
THE HIGHER BET, THE HIGHER CHANCES TO TRIGGER OP JACKPOT FEATURE.

## 六、遊戲規則 GAME RULES

▪ 選擇您想要玩的投注選項。
SELECT BET OPTION YOU WISH TO PLAY.

▪ 所有中獎金額乘以投注選項，除非是累積獎。
ALL WINS ARE MULTIPLIED BY BET OPTION EXCEPT PROGRESSIVES BONUSES.

▪ 所有符號必須由最左至右連續出現，方可獲獎；每條賠線只支付最高獎項。
ALL WINS ARE FROM LEFTMOST TO RIGHT ONLY; ONLY THE HIGHEST WIN IS PAID FOR EACH LINE.

▪ 當同局出現多個獲勝組合時，所有獎勵將加總為當局總獎勵。
WHEN WINNING WITH MULTIPLE SYMBOLS, ALL WINS ARE ADDED TO THE TOTAL WIN.

▪ 遊戲故障，所有賠付和遊戲都視為無效。
MALFUNCTION VOIDS ALL PAYS AND PLAYS.

▪ 若玩家在免費遊戲特色期間中斷遊戲，系統將會自動計算遊戲結果，並將得獎金額加入至餘額。
IF THE PLAYER LOSES CONNECTION DURING THE FREE GAME FEATURE, THE SYSTEM WILL CALCULATE THE GAME RESULTS AUTOMATICALLY AND THE WINNING PRIZES WILL BE ADDED INTO THE BALANCE.
